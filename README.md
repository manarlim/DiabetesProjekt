# Diabetes Prediction Project

## Description
This project predicts whether a person has diabetes based on health data such as Glucose, Blood Pressure, BMI, Age, etc.  
It uses machine learning with a RandomForest model and includes an interactive program where you can enter values to get a prediction.

## Features
- Interactive prediction: enter health data and get a prediction.
- Uses a trained RandomForest model for classification.
- Preprocessing with StandardScaler to normalize data.
- Clear step-by-step Jupyter Notebook included.

## Technologies
- Python 3
- Pandas, Numpy
- Scikit-learn
- Jupyter Notebook
